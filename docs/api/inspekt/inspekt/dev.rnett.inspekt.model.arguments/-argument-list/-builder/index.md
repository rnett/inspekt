# Builder

[common]\
class [Builder](index.md)

A builder to assemble an [ArgumentList](../index.md) for a set of parameters. Parameters which are not set will use their default values if they have one. Successfully building an [ArgumentList](../index.md) guarantees that values are provided for all parameters without default values.
