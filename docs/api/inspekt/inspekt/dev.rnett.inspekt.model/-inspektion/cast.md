# cast

[common]\
fun [cast](cast.md)(value: [Any](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin/-any/index.html)): [T](index.md)

Casts [value](cast.md) to this class. This is the same as using the `as` operator. Can be checked ahead of time with [isInstance](../../../../inspekt/dev.rnett.inspekt.model/-inspektion/--root--.md) (aka `is`), and done safely with [safeCast](../../../../inspekt/dev.rnett.inspekt.model/-inspektion/--root--.md) (aka `as?`).
