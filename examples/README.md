# Inspekt Examples

This directory contains examples of how to use Inspekt.

## Examples

- [hello-world](./hello-world): A simple JVM project using Inspekt.
- [multiplatform](./multiplatform): A Kotlin Multiplatform project using Inspekt in `commonMain`.

## Running the examples

To run the examples, use the provided Gradle wrapper:

### Hello World (JVM)

```bash
./gradlew :hello-world:run
```

### Multiplatform

```bash
./gradlew :multiplatform:jvmRun
```
